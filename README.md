### Blanche

An unofficial port of VS Code's Blanche (https://github.com/shytikov/blanche)

![Blanche](screenshots/blanche.png)

## About

Blanche is a clean, minimalist theme with a light color scheme designed for comfortable coding. This Zed extension brings the elegant Blanche theme from VS Code to Zed, maintaining its signature aesthetic while taking advantage of Zed's modern editor capabilities.
